# Liyaan Shopping – Women’s Clothing E-Commerce Website

## 📌 Project Overview

Liyaan Shopping is a web-based e-commerce platform designed for women’s clothing. The system allows users to browse, search, and purchase dresses online, while administrators manage products, inventory, and customer orders. The platform focuses on providing a smooth and user-friendly shopping experience with efficient backend management.


## 🎯 Features

### 👤 User Features

* User registration and login
* Browse and search products
* Filter and sort products (price, popularity, newest)
* View product details and stock availability
* Add/remove items to cart
* Update cart quantity
* Wishlist management (add/remove/view/share)
* Place orders and view order history
* Cancel orders and submit return requests
* Submit and view product reviews
* Contact customer support


### 🛒 Shopping Features

* Real-time stock visibility (size & color)
* Low-stock alerts (e.g., "Only 2 left")
* Secure checkout process
* Multiple payment method selection
* Order confirmation system


### 🧑‍💼 Admin Features

* Admin login
* Add, update, and delete products
* Manage product inventory
* View and manage customer orders
* Update order status
* View sales reports


## 🏗 System Architecture

The system follows a **Layered Architecture**, consisting of:

* **Presentation Layer** – User interface (frontend)
* **Application Layer** – Handles requests and user interactions
* **Business Logic Layer** – Processes core operations (cart, orders, payments)
* **Data Access Layer** – Communicates with the database
* **Database Layer** – Stores users, products, orders, and reviews
